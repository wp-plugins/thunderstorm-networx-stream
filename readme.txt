== Description ==

Ermöglicht das Anzeigen deiner Aktivität bei thunderstorm-networx auf deinem WordPress-Blog.

Hinweis: Um das Plugin nutzen zu können, benötigst du einen Account bei thunderstorm-networx.de. Klicke <a href="http://www.thunderstorm-networx.de/register">hier</a>, um dich kostenlos zu registrieren.

== Installation ==

Methode 1:
1. Gehe zu 'Plugins->Installieren' und suche nach 'thunderstorm-stream'
2. Klicke auf 'Jetzt installieren' und aktiviere das Plugin anschließend.
3. Registrieriere dich unter <a href="http://www.thunderstorm-networx.de/register">http://www.thunderstorm-networx.de/</a>.
4. Gehe im Menü auf 'thunderstorm-stream' und konfiguriere dein Plugin.
5. Aktiviere das Widget 'thunderstorm-stream' in der gewünschten Sidebar.

Methode 2:
1. Lade die Datei 'thunderstorm-stream.zip' bei WordPress.org herunter
2. Entpacke sie in den Ordner 'thunderstorm-stream' und laden diesen in den Order 'wp-content/plugins' deiner WordPress-Installation hoch.
3. Gehe nun zu 'Plugins' und aktiviere das Plugin
4. Registrieriere dich unter <a href="http://www.thunderstorm-networx.de/register">http://www.thunderstorm-networx.de/</a>.
5. Gehe im Menü auf 'thunderstorm-stream' und konfiguriere dein Plugin.
6. Aktiviere das Widget 'thunderstorm-stream' in der gewünschten Sidebar.
