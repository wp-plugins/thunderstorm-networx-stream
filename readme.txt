== Description ==

Ermöglicht das Anzeigen deiner Aktivität bei thunderstorm-networx auf deinem WordPress-Blog.

== Installation ==

1. Gehe zu 'Plugins->Installieren' und suche nach 'WP-Social'
1. Klicke auf 'Jetzt installieren' und aktiviere das Plugin anschließend.
1. Gehe im Menü auf 'thunderstorm-stream' und konfiguriere dein Plugin
